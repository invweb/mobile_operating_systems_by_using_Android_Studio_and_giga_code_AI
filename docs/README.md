# Screenshot

This is a placeholder for the app screenshot.

You can add your screenshot here by:
1. Running the app on an emulator or device
2. Taking a screenshot (File > Save As Screenshot in Android Studio)
3. Saving it as `app_screenshot.png` in this folder

Or use this command to capture:
```bash
adb shell screencap -p /sdcard/app_screenshot.png
adb pull /sdcard/app_screenshot.png docs/app_screenshot.png
```
